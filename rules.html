Playing Paritaire

Playing paritaire is a simple as it sounds because you need no login or password! But if you have some questions anyhow, you can read the FAQ below.

What are the game rules?

Paritaire is a game you might already know as reversi or othello and follows the same rules: Your main goal is to have more tokens in the end than your opponent.

But how can I reach this goal?
Every game starts with four crossed tokens. Now both players can set their tokens alternately by following these two rules:
  1. You can only put your token beside* an opposing token.
  2. You can only put it there if there is at least one of your tokens in an at it arising row*.
As a result of your move all opposing tokens that are bounded in a straight row by your new token and your other tokens will change their color and become yours.
If a player could not set any token the other player has another move.

How does the game end?
The game ends if there is no possibility left to place any token. That player that has the majority of tokens will win the game.

* horizontal, vertical & diagonal

What game modes are available?

On setting up a game you are asked to choose between different game modes. At the moment you can choose between the online and the local mode:

  • Online
    You can play via Internet on two devices. The only thing you have to do is share your game and wait for your opponent to join it.
    See also how to share and how to join games.

  • Local
    If you do not have or do not want to play on two devices you can also play on on device.

How can I start a game?

Settings
To start a new game session click the new button in the navigation bar to get to the set up options where you can set the different game parameters below.
When you finished your settings you can start the game by pressing the "Ready!" button which leads you to the play page. If you have chosen the local mode you can start the game immediatly, if you have chosen the online mode, you have to wait until your opponent joins the session.

Opponent
To invite a friend to play online with you, you can press the "Session Link" button to copy and send the Link. If you want to play with an unknown opponent press the "Publish" button to give your session a name and publish it an the main page.

Parameters
  • Field Size
    The field size ist set to 8x8 by default which is a good size for each diplay size (screens, tablets and smartphones).

  • Game Mode

  • Colors
    Before starting the players can choose a color they identify with during the game.

How can I join a game?

In general you can join each game by opening the session link in a browser.

How do I get a Session Link?
You can get your session link by three drifferent ways:

  • If you already have an opponent, but no session, you have to start a new one.

  • If your opponent already created a new session, he just has to tell you the session link. He can get it by pressing the "Session Link" button.

  • If you have no opponent you can join a public session by clicking on one. You can find all current public sessions under "Public" in the navigation bar or on the start page.
